Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element are:
document.getElementById();
document.getElementsByClassName();
document.getElementsByTagName();
document.querySelector();
document.querySelectorAll();

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHtml: When we use innerHTML property it returns html tag and the text content.

innerText: When we use innerText property it returns only the text content.

textContent: It is almost similar to innerText. innerText ignores texts which are hidden but textContent returns all the text content.

Question-3: What is event delegation in the DOM?

Event delegation is the process of selecting an element and using event bubbling to shorten the code and having the parent of that element do the work.

Question-4: What is event bubbling in the DOM?

When we target an element, event capturing goes to the target and propagates child to parent. When it propagates it runs all events reversely, it is called event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

>We can create element using: document.createElement();

>We can add elements using: appendChild();

>We can remove using: remove() or removeChild();
